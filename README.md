> [!note]
> 一个简洁的浅色主题，**[下载地址](https://github.com/caolib/theme.typora.io/releases)**
> **使用方法**：文件 → 偏好设置 → 外观 → 打开主题文件夹 → 将下载的2个css文件都粘贴到这个目录然后重新启动typora
> 
> 如果要自行修改，建议修改`onelight.user.css`文件，将你自己的样式放在这个文件中
